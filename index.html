Algorithmic Trading Bot

Creating an automatic Forex trading system that uses different trading strategies such as a Simple Moving Average or Relative Strength Index to buy/sell on given signals. I started this side project as a way to better understand automation, practice scripting, and algorithm writing.

Job Search Web Scraper

Scripting a job search site web scraper that extracts metadate from job boards and exports them to Excel to automate my job search. I am currently trying to research ways on pulling data from multiple sites and pages with the use of ASYNC IO.

PDF Circuit Annotator

Developed a Python automation tool during my extended internship. The task of finding structure damage and matching its ID to the structure data takes about one week manually. This tool cuts the time down to ~10 seconds while pulling structure data for an entire circuit.

Pi-Hole: A Black Hole for Internet Advertisements

Implemented a network-wide ad blocker using a Raspberry Pi 3 B to prevent a DNS query for advertisement/malware domain from returning the correct response by rerouting network DNS to the Pi-Hole static IP. The device handles all DNS requests generated from a network.

ChargeMe Mobile Application

Designed and developed a bill splitting mobile application using React Native and Expo/XCode for iOS. Implemented user login/account creation and assisted in structuring the realtime Firebase database.

Nominated for the CSULB Best Project Competition 2019 by faculty.

HTTPS Server & RSA Ransomware

Built and Amazon EC2 instance with NGINX as a reverse proxy server. Used Let's Encrypt to set up HTTPS certification by testing the TLS configuration with SSL Labs. As a second part to the project, I scripted RSA ransomware in Python that generates encrypted files in a directory.
